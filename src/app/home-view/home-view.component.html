Deferred Loading

When using deferred loading, you should keep in mind, that screenreaders probably don't notice changes once the deferred block is loaded when already in focus.

on timer

@defer (on timer(10000)) { } @placeholder() { Placeholder... } @loading(minimum 1000) { Loading... } @error { Error... }

on viewport

@defer (on viewport) { } @placeholder(minimum 3000) {
