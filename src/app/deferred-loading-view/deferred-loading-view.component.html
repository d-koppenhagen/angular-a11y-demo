Deferred Loading

When using deferred loading, you should keep in mind, that screenreaders probably don't notice changes once the deferred block is loaded when already in focus.

on timer

@defer (on timer(5000)) { } @placeholder() { Placeholder... } @loading(minimum 3000) { Loading... } @error { Error... }

on viewport

Please scroll down below the bottom border to see deferred loading on viewport in action.
